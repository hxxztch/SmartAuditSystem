# 高校一体化智慧审计系统 - SmartAuditSystem

基于 Spring Boot + Vue 3 构建的高校一体化智慧审计平台。

## 项目结构

```
SmartAuditSystem/
├── frontend/          # 前端 - Vue 3 + Vite
└── backend/           # 后端 - Java Spring Boot + Maven
```

## 快速启动

### 前端

```bash
cd frontend
npm install
npm run dev
```

### 后端

用 IDE 打开 `backend/`，运行 Maven 项目即可。

## 技术栈

- **前端**：Vue 3, Vite, Element Plus
- **后端**：Spring Boot, MyBatis, H2 Database
- **数据库**：H2 (嵌入式)
